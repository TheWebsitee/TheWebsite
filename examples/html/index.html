Tables

I need some tables! Here's how you can put your text into a table. To style your table it make it look better you will have to refer to this section on the css page.

But it is pretty easy. So first you need to put your table tag thing in. like this:

<table>

Next, you have to put the "tr" tag. Which means table row. follow that up with some "th" tags. Which means table header. Make sure you close those tags aswell. Always close your tags! If you use visual studio or really any code editing software other than notepad it should automatically close your tags for you.

It should look like this right now:

<table>
        <tr>
                <th>Header 1</th>
        </tr>
</table>

Next up is table data Aka "td aka the stuff your going to put in your table. So you will start it off by putting your "tr" tag then your "td tag. Then you can put what the words should say there.

You would continue putting "td" tags for each column you have. It should now look like this.

<table>
        <tr>
                <th>Header 1</th>
                <th>Header 2</th>
        </tr>
        <tr>
                <td>Hi</td>
                <td>Ello</td>
        </tr>
</table>

Here is the end result.

Header 1 Header 2
Hi Ello
