CSS

Here you got css stuff. I decided not to do a table or anything because all you need to know is the tags.

Tags

Here is the tags. This is the stuff you would put in your css elements.

Each element should look like this and I included a tag example (backround-color):

body {
        backround-color: black;
}

Border

Here is where all the border related tags are.

  • Border
  • Border radius

This is going to be your border. Here is the code example.

body {
        backround-color: black;
        border: 2px solid gray;
}

Border radius

This is going to be your border radius. Here is the code example.

body {
        backround-color: black;
        border: 2px solid gray;
        border-radius: 16px;
}

Text

Here is where all the Text related tags are.

  • Font Family

Font family

This is going to be your font family. Here is the code example.

body {
        backround-color: black;
        border: 2px solid gray;
        border-radius: 16px;
        font-family: serif;
}

Font weight

This is going to be your font weight. You can set how thick or thin the text is. By using anywhere between 100-900 Here is the code example.

body {
        backround-color: black;
        border: 2px solid gray;
        border-radius: 16px;
        font-family: serif;
        font-weight: bold;
}
