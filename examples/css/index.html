Body

Here is the body stuff. There are many different things you can change. So anything you set in this bracket will apply to any text etc in the body bracket in your html.

So First let's start with "backround-color". This is literally what it says it is. It changes the backround color, for example if your set yours to black your backround will be black. Now this doesn't control the backround color of the whole page. That would be controlled in the html bracket you can find that Here

Anyways, so here is what the line of code would look like if you were to set your backround to black:

body {
        backround-color: black;
}
